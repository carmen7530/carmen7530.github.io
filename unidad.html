Unidad 3. Estructuras Lineales

3.1 Pilas

Las pilas son estructuras de datos lineales que siguen el principio LIFO (Last In, First Out).

3.1.1 Representación en memoria

Se representan usando arreglos o listas enlazadas, controladas por un puntero o índice para la cima.

3.1.2 Operaciones básicas

  • Push: Insertar un elemento en la cima.
  • Pop: Eliminar el elemento en la cima.
  • Peek: Visualizar el elemento en la cima sin eliminarlo.

3.1.3 Aplicaciones

  • Gestión de llamadas a funciones (pila de llamadas).
  • Evaluación de expresiones (notación postfija y prefija).
  • Deshacer acciones en aplicaciones.

3.2 Colas

Las colas siguen el principio FIFO (First In, First Out).

3.2.1 Representación en memoria

Se implementan con arreglos o listas enlazadas, gestionando dos punteros: frente y final.

3.2.2 Operaciones básicas

  • Enqueue: Insertar un elemento al final.
  • Dequeue: Eliminar el elemento al frente.
  • Peek: Visualizar el elemento al frente sin eliminarlo.

3.2.3 Tipos

  • Cola simple.
  • Cola circular.
  • Bicola (Deque).

3.2.4 Aplicaciones

  • Gestión de tareas en sistemas operativos.
  • Procesamiento de datos en redes.
  • Simulación de colas (bancos, supermercados).

3.3 Listas

Las listas son colecciones de elementos donde el orden y la posición son importantes.

3.3.1 Operaciones básicas

  • Inserción: Agregar elementos en cualquier posición.
  • Eliminación: Quitar elementos.
  • Recorrido: Desplazarse a través de los elementos.
  • Búsqueda: Encontrar elementos específicos.

3.3.2 Tipos

  • Lista simplemente enlazada.
  • Lista doblemente enlazada.
  • Lista circular.

3.3.3 Aplicaciones

  • Almacenamiento dinámico de datos.
  • Implementación de otras estructuras como pilas y colas.
  • Sistemas de navegación.

Código de ejemplo

class Pila {
    private int[] stackArray;
    private int top;
    
    public Pila(int size) {
        stackArray = new int[size];
        top = -1;
    }
    
    public void push(int value) {
        if (top < stackArray.length - 1) {
            stackArray[++top] = value;
        }
    }
    
    public int pop() {
        return top >= 0 ? stackArray[top--] : -1;
    }
}

Unidad 4. Estructuras No Lineales

4.1 Árboles

Los árboles organizan datos jerárquicamente.

4.1.1 Clasificación

  • Árbol general.
  • Árbol binario.
  • Árbol binario de búsqueda.
  • Árbol balanceado.

4.1.2 Operaciones básicas

  • Inserción.
  • Eliminación.
  • Recorridos (Preorden, Inorden, Postorden).

4.1.3 Aplicaciones

  • Búsquedas rápidas.
  • Representación de expresiones.
  • Árboles de decisión.

4.2 Grafos

Los grafos son estructuras de nodos y aristas.

4.2.1 Representación

  • Matriz de adyacencia.
  • Lista de adyacencia.

4.2.2 Operaciones básicas

  • DFS.
  • BFS.
  • Adición/Eliminación de nodos y aristas.

Código de ejemplo

class Nodo {
    int dato;
    Nodo izquierda, derecha;

    public Nodo(int dato) {
        this.dato = dato;
        izquierda = derecha = null;
    }
}
